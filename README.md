# Why This Repository Exists

My old laptop was struggle with windows 10. Very very slow. 
It becomes so quick i change it to Linux mint XFCE. 

At first, I don't know anything about Linux and make many mistake.

But little by little, I started understanding:

- how Linux manages memory
- keep systems clean and quick.
- how lightweight environments matter
- how developers actually use Linux daily

This repository is not a perfect tutorial.

It is a growing collection of:
- notes
- experiments
- fixes
- optimization tricks
- setup guides
- mistakes
- lessons learned
